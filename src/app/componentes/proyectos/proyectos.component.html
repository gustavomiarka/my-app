Practica Profesionalizante I Geo Pet

Fecha: Julio 2021 - Diciembre 2021

Proyecto integral de la asignatura Práctica Profesionalizante, ciclo 2021 de la Tecnicatura Superior en Desarrollo Web y Aplicaciones Digitales del Instituto Superior Politécnico Córdoba. En este proyecto si incoporaron los saberes adquiridos en las materias de Programación 1, Programación Web 1 y Base de datos. Se implementó HTML5 para maquetado, CSS3 para implementar estilos, Javascript para funcionalidades y Bootstrap. Este proyecto comprende una App que permite la localización de mascotas usando el sistema GPS. Se monitorea la ubicación en tiempo real a través de un dispositivo integrado al collar del animal.
